## ⚡ Quick Start (3 Steps)

### 1. Open Your Browser
Go to: **http://localhost:3000**

### 2. Start the Quiz
Click **"Start Your Assessment"** button

### 3. Generate Your Course
Answer 8 questions → Click "Generate Course" → Wait 30-45 seconds

## 🛠️ Server Commands

### Stop the Server
Press `Ctrl + C` in the terminal

### Restart the Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
npm start
```
